### 注意事项

- 1.WXGiftMoney仅在【微信群聊界面】功能有效。
- 2.WXGiftMoney最低兼容到微信6.6.5版本(原作者会对微信之后的版本进行兼容)。
- 3.建议打开后WXGiftMoney后，将应用放到后台，不要关掉应用（某些手机关掉应用时会结束掉辅助服务）。
